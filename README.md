# GetLando!

A [2048](http://gabrielecirulli.github.io/2048/) clone for **Formula 1** fans. Swipe and merge F1 drivers to reach the almighty Lando Norris tile!

[Play it here!](https://dcarley04.github.io/GetLando/)

## The tile ladder

| Tile | Driver |
|------|--------|
| 2 | Arvid Lindblad (Racing Bulls) |
| 4 | Liam Lawson (Racing Bulls) |
| 8 | Pierre Gasly (Alpine) |
| 16 | Isack Hadjar (Red Bull) |
| 32 | Max Verstappen (Red Bull) |
| 64 | George Russell (Mercedes) |
| 128 | Lewis Hamilton (Ferrari) |
| 256 | Oscar Piastri (McLaren) |
| 512 | Kimi Antonelli (Mercedes) |
| 1024 | Charles Leclerc (Ferrari) |
| **2048** | **Lando Norris (McLaren)!** |
| Beyond | Super Lando |

Click the Lando emblem in the title bar to toggle a McLaren / Ferrari / Mercedes / Red Bull background.
